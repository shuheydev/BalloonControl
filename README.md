# BalloonControl

吹き出し型のコントロールです.
中にテキストを表示します.

<image src='https://user-images.githubusercontent.com/43431002/71499873-da92f380-28a5-11ea-892b-36bcc62f32b1.png' width=350/>

iOSでは動作,見た目の確認をしていません.

## プロパティ
### Text
表示する文字列.
### BalloonColor
吹き出しの色.
### CornerRadius
吹き出しの角の丸み.範囲:0~20
### MouthDirection
吹き出しの口の方向(Right,Left)
### TextColor
文字色.
